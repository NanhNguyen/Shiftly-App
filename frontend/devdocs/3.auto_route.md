# Định tuyến thông minh với AutoRoute

Flutter mặc định dùng `Navigator.push` với tên path, nhưng sẽ khổ nếu ta truyền tham số (như ID bài viết, obj user) vì phải ép kiểu.
**ShiftFlow** sử dụng `auto_route` để đảm bảo code navigation rõ ràng, sinh code tự động và bắt lỗi ngay khi gõ.

## 1. Cách khai báo một màn hình (Screen) mới

Để một Màn hình có thể điều hướng được, chỉ cần gắn `@RoutePage()` vào phía trên Class của màn hình, thay vì `StatelessWidget`:

```dart
import 'package:auto_route/auto_route.dart';

@RoutePage()
class ManagerRequestPage extends StatelessWidget {
  const ManagerRequestPage({super.key});

  // ...
}
```

## 2. Khai báo vào file trung tâm

Mở file `frontend/lib/ui/router/app_router.dart`, thêm `AutoRoute` vào danh sách. **CHÚ Ý**: Tên route sẽ luôn là `<TênClass>Route.page` được sinh tự động.

```dart
@AutoRouterConfig()
class AppRouter extends RootStackRouter {
  @override
  List<AutoRoute> get routes => [
    AutoRoute(page: LoginRoute.page, initial: true), // initial = trang đầu tiên
    AutoRoute(page: MainRoute.page),
    AutoRoute(page: NotificationRoute.page),
    AutoRoute(page: ManagerRequestRoute.page), // <- Thêm mới vào đây
  ];
}
```

## 3. Sinh code (Bắt buộc)
Lại là lệnh thần thánh, bạn phải chạy nó mỗi khi gắn `@RoutePage` mới:
```bash
dart run build_runner build --delete-conflicting-outputs
```
Nó sẽ sinh ra file `app_router.gr.dart`.

## 4. Cách chuyển trang (Navigation)

### Chuyển sang màn mới (Có thể bấm back)
```dart
context.router.push(const NotificationRoute());
// Hoặc ngắn gọn
context.pushRoute(const NotificationRoute());
```

### Chuyển màn và truyền tham số
Nếu màn hình bạn cần tham số (VD constructor có `final bool isEdit`):
```dart
// AutoRoute tự hiểu bạn cần truyền isEdit:
context.pushRoute(ScheduleFormRoute(isInitialRecurring: true));
```

### Quay về màn cũ
```dart
context.router.pop();
```

### Quay về trang chủ (Root) xoá hết lịch sử
```dart
context.router.popUntilRoot();
```

### Chuyển màn hình nhưng xoá màn hiện tại (Thay thế)
```dart
context.router.replace(const MainRoute());
```

## 5. Các Lỗi thường gặp
- Ghi tên màn hình là `NotificationPage` nhưng code không nhận `NotificationRoute`? Nguyên nhân: Bạn quên chạy `build_runner`.
- Không gọi được `context.router`? Hãy import gói của nó: `import 'package:auto_route/auto_route.dart';`
