# TableCalendar & Định dạng ngày tháng (Intl)

Trái tim của tính năng Schedule tại **ShiftFlow**! 

## 1. Intl - Format ngày giờ cho đúng

`intl` cực kỳ hữu hiệu khi chuyển qua lại giữa JSON String từ Server (`"2026-03-01T...Z"`) về kiểu Date hiển thị.

```dart
// Format cho User xem:
import 'package:intl/intl.dart';

final date = DateTime.now(); // 2026-02-27 ...
final formatType = DateFormat('dd/MM/yyyy').format(date); // 27/02/2026 
```

## 2. Table Calendar
Là package dùng ở màn `schedule_page.dart` (hiệu ứng trượt, chọn theo tuần, làm dots cực kỳ dễ). 

### 4 Khái niệm cốt lõi:
- **`focusedDay`** (Ngày đang xem/Lật trang ở đó) -> *Nếu gán nó là Hôm Nay => trang đầu tiên là Tháng hiện tại.*
- **`firstDay` / `lastDay`**: Phạm vi tối đa của quyển lịch (VD: Từ 2020 đến 2030).
- **`selectedDay`**: Ngày bị chấm tô màn xanh. (Phải xài setState để báo lịch vẽ lại ngày được quét).
- **`calendarFormat`**: Có 2 dạng (Week và Month). Có thể tuỳ biến UI như ta đang làm ở `schedule_page.dart`.

### Hàm EventLoader
Khi bạn chạm tay qua tháng sau, Lịch sẽ gọi ngầm một hàm lặp **trên từng ngày** của tháng đó, qua tham số `eventLoader`.
Bạn có list Schedule `A`, nó sẽ duyệt mỗi ngày, lấy trong `A` ra những Schedule thuộc về Ngày đó, nếu trả ra >0 cái => Nó tự chấm `markerBuilder` (dấu chấm dưới lịch Tháng).

```dart
TableCalendar(
  eventLoader: (day) => getSchedulesForNgay(day),
)

// Code tự viết: Lấy TẤT CẢ request thoả mãn trùng "ngày này"

    // Return all models match the date...
}
```

### Hàm CalendarBuilders
Thay đổi từng UI bé tí của Lịch thông qua các builder: `defaultBuilder`, `todayBuilder`, `selectedBuilder`, `markerBuilder`.

```dart
calendarBuilders: CalendarBuilders(
  // Thay thế ô hiển thị bằng Widget (ví dụ hiển thị chữ Sáng, Chiều)
  defaultBuilder: (context, day, focusedDay) => 
     Text("27"), 
  
  // Custom Dot (khi eventLoader trả về > 0 phần tử)
  markerBuilder: (context, date, events) {
      if (events.isEmpty) return null;
      return Container(width: 8, height: 8, color: Colors.blue);  
  }
)
```

## Quản lý UI ở màn Week View (Đại tu gần nhất)
Ở màn `schedule_page.dart` (Tab Nghỉ Định Kỳ - Week View), ta ẩn cái `marker` chấm gốc của `TableCalendar` đi, mà Custom toàn bộ bằng `defaultBuilder`. Tại đây:
- Kiểm tra số lượng Model Ca Sáng có bằng Ngày hiện tại hay không
- Kiểm tra số lượng Ca Chiều.
- Vẽ chữ SA, CH kèm số lượng tuơng ứng.
- Đặt `rowHeight: 250` để lịch đủ diện tích in hình thay vì bị thu hẹp.
