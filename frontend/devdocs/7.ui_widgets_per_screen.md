# 🎨 UI Widgets & Libraries – Chi tiết từng màn hình

> Tài liệu này mô tả **thư viện**, **widget**, và **các argument quan trọng** được dùng tại mỗi màn hình trong app ShiftFlow.

---

## 📦 Thư viện UI chính

| Package | Version | Mục đích |
|---|---|---|
| `flutter/material.dart` | SDK | Material Design widgets (Scaffold, AppBar, Card, ...) |
| `flutter/cupertino.dart` | SDK | iOS-style widgets (CupertinoDatePicker) |
| `flutter_bloc` | ^9.1.0 | State management – BlocBuilder, BlocConsumer, BlocProvider |
| `table_calendar` | ^3.2.0 | Hiển thị lịch dạng tuần / tháng |
| `intl` | ^0.20.2 | Định dạng ngày tháng (DateFormat) |
| `auto_route` | ^9.2.2 | Điều hướng (context.router.push, context.pushRoute) |
| `google_fonts` | ^8.0.2 | Custom fonts (hiện dùng system font mặc định) |

## 📱 Màn hình 1: `LoginPage` (`login/login_page.dart`)

**Loại widget:** `StatefulWidget`  
**State manager:** `BlocProvider` + `BlocConsumer<LoginCubit, LoginState>`

### Cấu trúc cây widget

```
Scaffold
└── Container (gradient background)
    └── Center
        └── SingleChildScrollView
            └── Card
                └── Padding
                    └── Column
                        ├── Icon (calendar_month)
                        ├── Text (app name)
                        ├── TextField (email)
                        ├── SizedBox
                        ├── TextField (password)
                        ├── SizedBox
                        └── SizedBox > ElevatedButton (login)
```

### Chi tiết widgets

#### `Scaffold`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `backgroundColor` | `Colors.blue.shade900` | Màu nền fallback khi gradient load |
| `body` | `Container` | Nội dung chính |

#### `Container` (background gradient)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `decoration` | `BoxDecoration(gradient: LinearGradient(...))` | Gradient từ blue.900 → blue.700, hướng topCenter → bottomCenter |

#### `Card`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `elevation` | `8` | Đổ bóng, tạo cảm giác nổi |
| `shape` | `RoundedRectangleBorder(borderRadius: BorderRadius.circular(24))` | Bo góc 24px |

#### `TextField` (email)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `controller` | `_emailController` | TextEditingController để lấy giá trị |
| `decoration.labelText` | `AppStrings.email` | Label nổi lên khi focus |
| `decoration.prefixIcon` | `Icon(Icons.email_outlined)` | Icon email ở bên trái |
| `decoration.border` | `OutlineInputBorder(borderRadius: BorderRadius.circular(12))` | Border bo góc 12px |

#### `TextField` (password)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `controller` | `_passwordController` | TextEditingController để lấy giá trị |
| `obscureText` | `state.obscurePassword` | Ẩn ký tự khi true |
| `decoration.suffixIcon` | `IconButton` với `Icons.visibility` / `Icons.visibility_off` | Toggle hiện/ẩn mật khẩu |

#### `ElevatedButton` (login button)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `onPressed` | `null` khi loading, gọi `LoginCubit.login()` khi sẵn sàng | Disable button khi đang loading |
| `style.backgroundColor` | `Colors.blue` | Màu nền button |
| `style.foregroundColor` | `Colors.white` | Màu text/icon trong button |
| `style.shape` | `RoundedRectangleBorder(borderRadius: 12)` | Bo góc |
| `child` | `CircularProgressIndicator` (loading) hoặc `Text` | Switch giữa loading và text |

#### `BlocConsumer<LoginCubit, LoginState>`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `listener` | Kiểm tra `state.status` | Khi success → navigate, khi error → hiện SnackBar |
| `builder` | Rebuild UI theo state | Truy cập `state.obscurePassword`, `state.status` |

#### `SnackBar` (error)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `content` | `Text(state.errorMessage)` | Hiện thông báo lỗi từ cubit |

## 📱 Màn hình 2: `MainPage` (`main/main_page.dart`)

**Loại widget:** `StatefulWidget`  
**State manager:** `MultiBlocProvider` + `BlocBuilder<MainCubit>` + `BlocBuilder<NotificationCubit>`

### Cấu trúc cây widget

```
MultiBlocProvider
└── BlocBuilder<NotificationCubit>
    └── BlocBuilder<MainCubit>
        └── InAppNotificationOverlay (custom widget)
            └── Scaffold
                ├── body: IndexedStack
                │   └── [HomePage, SchedulePage, StatusPage/ManagerPage, ProfilePage]
                └── bottomNavigationBar: BottomNavigationBar
```

### Chi tiết widgets

#### `MultiBlocProvider`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `providers` | `[BlocProvider.value(value: _mainCubit), ...]` | Cung cấp 3 cubits: MainCubit, HomeCubit, NotificationCubit |

#### `IndexedStack`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `index` | `state.currentIndex` | Tab đang hiển thị |
| `children` | List các page widget | Giữ state của tất cả tab (không rebuild khi switch) |

> **Lý do dùng IndexedStack thay ListView:** IndexedStack giữ nguyên widget tree và state của các tab, tránh re-initialize mỗi khi người dùng chuyển tab.

#### `BottomNavigationBar`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `currentIndex` | `state.currentIndex` | Tab đang active |
| `onTap` | Gọi `MainCubit.setIndex(index)` | Đổi tab, trigger reload Schedule nếu cần |
| `type` | `BottomNavigationBarType.fixed` | Hiển thị label cho tất cả item (không ẩn khi nhiều tab) |
| `selectedItemColor` | Màu theo role (blue/deepPurple/teal) | Active tab color |
| `unselectedItemColor` | `Colors.grey` | Inactive tab color |
| `iconSize` | `30` | Kích thước icon |
| `selectedLabelStyle` | `TextStyle(fontSize: 14, fontWeight: bold)` | Label tab active |
| `unselectedLabelStyle` | `TextStyle(fontSize: 13)` | Label tab inactive |
| `items` | `List<BottomNavigationBarItem>` thay đổi theo role | Khác nhau cho INTERN/MANAGER/HR |

#### `BottomNavigationBarItem`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `icon` | `Icon(Icons.home_outlined)` | Icon khi tab chưa active |
| `activeIcon` | `Icon(Icons.home)` | Icon khi tab đang active (fill style) |
| `label` | `AppStrings.home` | Nhãn tab |

## 📱 Màn hình 3: `HomePage` (`home/home_page.dart`)

**Loại widget:** `StatelessWidget`  
**State manager:** `BlocBuilder<HomeCubit, HomeState>`

### Cấu trúc cây widget

```
BlocBuilder<HomeCubit>
└── Scaffold
    ├── AppBar
    │   ├── title: Text
    │   └── actions: [Stack > IconButton (🔔) + Positioned badge, SizedBox]
    ├── body: RefreshIndicator
    │   └── SingleChildScrollView
    │       └── Column
    │           ├── _buildHeader        → Container (gradient card)
    │           ├── _buildTodayStatus   → Container (gradient card)
    │           ├── _buildQuickActions  → Row of InkWell items
    │           ├── _buildQuickStats    → Row of Expanded cards
    │           └── Text (recent updates)
    └── floatingActionButton: FloatingActionButton (chỉ INTERN)
```

### Chi tiết widgets

#### `AppBar`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `title` | `Text(AppStrings.scheduleOverview)` | Tiêu đề màn hình |
| `actions` | `[Stack(IconButton + badge), SizedBox]` | Nút thông báo với badge |

#### `Stack` (notification badge)
Dùng `Stack` + `Positioned` để đặt badge số đỏ lên trên `IconButton`:
```dart
Stack(
  children: [
    IconButton(icon: Icon(Icons.notifications_none, size: 32), ...),
    Positioned(
      right: 8, top: 8,
      child: Container(
        // Badge đỏ hiển thị tổng số (unread + pending)
        decoration: BoxDecoration(color: Colors.red, borderRadius: BorderRadius.circular(10)),
        constraints: BoxConstraints(minWidth: 16, minHeight: 16),
        child: Text('$totalBadge', ...),
      ),
    ),
  ],
)
```

#### `RefreshIndicator`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `onRefresh` | `() => context.read<HomeCubit>().loadData()` | Pull-to-refresh gọi lại API |
| `child` | `SingleChildScrollView` | Nội dung có thể scroll |

#### `SingleChildScrollView`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `physics` | `AlwaysScrollableScrollPhysics()` | Bắt buộc scroll dù content ngắn (để pull-to-refresh hoạt động) |

#### `FloatingActionButton` (chỉ INTERN/EMPLOYEE)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `backgroundColor` | `Colors.blue.shade700` | Màu nền FAB |
| `foregroundColor` | `Colors.white` | Màu icon |
| `child` | `Icon(Icons.add)` | Icon dấu "+" |
| `onPressed` | Mở `showModalBottomSheet` | Hiện menu chọn loại nghỉ |

#### `showModalBottomSheet` (menu chọn loại nghỉ)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `shape` | `RoundedRectangleBorder(borderRadius: BorderRadius.vertical(top: Radius.circular(20)))` | Bo góc trên của sheet |
| `builder` | `Container > Column > [ListTile, ListTile]` | Menu 2 lựa chọn: Định kỳ / Đột xuất |

#### `ListTile` (trong BottomSheet)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `leading` | `Icon(Icons.repeat, color: Colors.blue)` | Icon phân biệt loại |
| `title` | `Text(AppStrings.recurringLeave)` | Nhãn lựa chọn |
| `onTap` | Navigate đến `ScheduleFormRoute` với param tương ứng | Điều hướng |

## 📱 Màn hình 4: `SchedulePage` (`schedule/schedule_page.dart`)

**Loại widget:** `StatefulWidget`  
**State manager:** `BlocProvider` (tạo `ScheduleCubit`) + `BlocBuilder<ScheduleCubit>`  
**Thư viện đặc biệt:** `table_calendar`, `intl`

### Cấu trúc cây widget

```
BlocProvider<ScheduleCubit>
└── Scaffold
    ├── AppBar (với actions: today button, refresh button)
    ├── body: DefaultTabController(length: 2)
    │   └── Column
    │       ├── [nếu Manager/HR] _buildSearchBar() → TextField
    │       ├── Container > TabBar (2 tab: Định kỳ / Đột xuất)
    │       └── Expanded > BlocBuilder
    │           └── TabBarView
    │               ├── _buildCalendarTab(isRecurring: true)  → Week view
    │               └── _buildCalendarTab(isRecurring: false) → Month view
```

### Chi tiết widgets

#### `DefaultTabController`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `length` | `2` | Số lượng tab |

#### `TabBar`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `indicatorColor` | `Colors.white` | Màu gạch dưới tab active |
| `indicatorWeight` | `3` | Độ dày gạch |
| `labelStyle` | `TextStyle(fontWeight: bold, color: white, fontSize: 18)` | Style tab active |
| `unselectedLabelStyle` | `TextStyle(fontWeight: bold, color: white60, fontSize: 17)` | Style tab inactive |

#### `TableCalendar` (thư viện `table_calendar`)
| Argument | Kiểu | Mô tả |
|---|---|---|
| `firstDay` | `DateTime(2020)` | Ngày đầu tiên có thể hiển thị |
| `lastDay` | `DateTime(2030)` | Ngày cuối cùng có thể hiển thị |
| `focusedDay` | `DateTime` (state) | Ngày đang được focus (điều hướng tháng/tuần) |
| `calendarFormat` | `CalendarFormat.week` (Định kỳ) / `CalendarFormat.month` (Đột xuất) | Dạng hiển thị lịch |
| `availableCalendarFormats` | `{week: 'Week', month: 'Month'}` | Các format cho phép switch |
| `locale` | `'vi'` | Ngôn ngữ tiếng Việt |
| `rowHeight` | `180` (week) / `80` (month) | Chiều cao mỗi hàng ngày |
| `selectedDayPredicate` | `(day) => isSameDay(_selectedDay, day)` | Xác định ngày được chọn |
| `onDaySelected` | Callback `(selectedDay, focusedDay)` | Cập nhật state khi bấm ngày |
| `onPageChanged` | Callback `(focusedDay)` | Cập nhật khi lật tháng/tuần |
| `eventLoader` | Callback trả về `List<ScheduleRequestModel>` | Dữ liệu event cho mỗi ngày |
| `calendarBuilders` | `CalendarBuilders(...)` | Custom các builder |

#### `HeaderStyle` (thuộc TableCalendar)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `formatButtonVisible` | `false` | Ẩn nút switch format |
| `titleCentered` | `true` | Căn giữa tiêu đề tháng/tuần |

#### `DaysOfWeekStyle` (thuộc TableCalendar)

## 📱 Màn hình 5: `ScheduleFormPage` (`schedule_form/schedule_form_page.dart`)

**Loại widget:** `StatelessWidget` (được wrap bởi `BlocProvider`)  
**State manager:** `BlocProvider` (tạo ScheduleFormCubit) + `BlocConsumer`  
**Thư viện đặc biệt:** `flutter/cupertino.dart` (CupertinoDatePicker), `intl`  
**Route param:** `isInitialRecurring: bool`

### Cấu trúc cây widget

```
BlocProvider<ScheduleFormCubit>
└── BlocConsumer<ScheduleFormCubit>
    └── Scaffold
        ├── AppBar (title thay đổi theo isRecurring)
        └── body: Container (gradient) > SingleChildScrollView > Column
            ├── _buildSectionTitle
            ├── [nếu Recurring] Row([_buildDateCard(start), Icon(→), _buildDateCard(end)])
            ├── [nếu Adhoc]    _buildMultiDatePicker
            ├── [nếu Recurring] _buildWeekdaySelector
            ├── _buildSectionTitle('CA')
            ├── _buildGlassDropdown (chọn ca)
            ├── _buildSectionTitle('GHI CHÚ')
            ├── TextField (description)
            └── ElevatedButton (submit)
```

### Chi tiết widgets

#### `AppBar`
| Argument | Giá trị | Mô tả |
|---|---|---|
| `title` | Định kỳ / Đột xuất theo `state.isRecurring` | Tiêu đề động |
| `backgroundColor` | `Colors.blue.shade700` | Màu AppBar |
| `foregroundColor` | `Colors.white` | Màu title và back button |

#### `_buildDateCard` (chọn ngày bắt đầu/kết thúc cho Recurring)
```dart
InkWell(
```

#### `CupertinoDatePicker` (trong BottomSheet)
| Argument | Kiểu | Mô tả |
|---|---|---|

#### `FilterChip` (chọn thứ trong tuần – Recurring mode)
| Argument | Giá trị | Mô tả |
|---|---|---|
| `label` | `Text('Th 2')` ... `Text('Th 6')` | Nhãn thứ |
